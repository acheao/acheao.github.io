SQL笔记

发表于 2019-01-26 | 分类于 笔记
第一部分
  • show databases;　　显示数据库列表
  • create database test;　　创建数据库test
  • show columns from user;　　显示列名及信息
  • select distinct username from user;　　查询并去除重复记录
  • select username from user limit 2;　　查询并显示2行记录
  • select username from user limit N,K;
    = select username from user limit K offset N;
    N表示第N+1列开始(包含第N+1行)，K条记录
  • select user.username from user;　　全限定名
  • select username from user order by columnsname;
    查询结果按照列columnsname排序
  • select username from user order by username desc;
    查询结果按照列columnsname倒序
第二部分
!= 不等于 　<>不等于　　>大于　　>=大于等于
between A and　B　介于A和B之间(包含A和B)
  • select * from user where id is null;　　is关键字
  • select * from user where id = 20 and price <= 1000;　　and操作符
  • select * from user where price > 200 or count <20;　　or操作符
  • select * from user where id = 100 or id = 201 and price >100;　　and优先级高于or
  • select * from user where (id = 100 or id = 201) and price >100;　　可以使用括号解决优先级问题
  • select * from user where id in (1,2,3,4);　　in操作符
  • select * from user where id not in (1,2,3);　　not操作符,对查询条件取反，可以和between，exists连用
  • select * from user where username like ‘lin%‘;　　like通配符过滤,区分大小写
  • select * from user where username like ‘%lin%‘;　　like通配符过滤%不能匹配null
  • select * from user where username like ‘_lin’；　　_匹配任意字符但是只匹配一个，%可以匹配任意数量(包括0个)
第三部分
  • select * from user where username regexp ‘lin’;　　正则表达式匹配包含lin字符的列，与like不同，like需要完全匹配才返回列，regexp只需部分匹配，并且不区分大小写
  • select * from user where usenamr regexp ‘.i’;　　. 正则表达式中匹配任意1个字符
  • select * from user where username regexp ‘lin|小’;　　| 逻辑或
  • select * from user where username regexp ‘100|200|300’;　　多个或并列
  • select * from user where username regexp binary ‘lin’;　　binary关键词使正则表达式匹配区分大小写
  • select * from user where username regexp ‘[123]lin’;　　匹配几个字符之一
    = select * from user where username regexp ‘1lin|2lin|3lin‘;
  • select * from user where username regexp ‘[^123]’;　　^字符集的否定，即除了123均匹配
  • [0123456789]等同于[0-9]　　　\\.　表示. 　　　\\\　表示\　　　\\_　表示_
  • [a-zA-Z0-9]任意字母和数字　　[:print:]任意可打印字符　　[:graph:]与[:print:]相同，但不包括空格　　
  • 重复元匹配*　+　？　{n}　{n,}　{n,m}
    *　　0个或多个匹配　　　+　　 1个或多个匹配 (等于{1，})　　?　　0个或1个匹配（等于{0，1}）
    {n}　　指定匹配的数目　　{n,}　　不少于指定数目的匹配　　{n,m}　　匹配数目的范围
  • 定位符^　$　[[:　[[:>:]]
    ^　　文本的开始或者集合的否定　　$　　文本的结尾　　[[:　　　词的开始　　[[:>:]]　　词的结尾
    select * from user where username regexp ‘^[0-9]’;　　匹配以数字开头的字段

    1.concat() 拼接字符串

2.rtrim() 去掉值右边的空格        
  ltrim() 去掉值左边的空格

  count(*) 返回总列数  
  count（username）返回某一列，除去null的行数
9.max() 返回某列的最大值  
  忽略值为null的行 如果列为文本则返回最后一行 
7.min() 返回某列的最小值  
  忽略值为null的行 如何列为文本则返回最前一行
8.sum() 返回某列值之和    
  忽略值为null的行
9.avg() count() max() min() 
  可以使用distinct区分不同,如count（distinct column）
